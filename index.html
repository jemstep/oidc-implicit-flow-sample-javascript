OpenID Connect Implicit Flow Sample - Javascript

This sample demonstrates the OpenId Connect Implicit flow. It is based on https://github.com/onelogin/onelogin-oidc-node

This flow is useful for single page applications as it does not require any server side interaction with the OpenId Connect provider.

Try it

After clicking login the user will be redirect to Jemstep's OpenID Provider to authenticate. Once authenticated they will be redirected back to this site and their profile details will be displayed below
